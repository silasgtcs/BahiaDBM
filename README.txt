 
BahiaDBM v1.0 (rev1)

- Software para geração de modelos lógicos, conceituais e físicos de bancos de dados.



Sistema de Versões:

- Novas versões são lançadas quando houverem modificações a nível de usuário (incluíndo bugs fatais).
- Novas revisões serão lançadas quando houverem modificações internas ao programa, estruturais, de código (incluíndo bugs mínimos).
